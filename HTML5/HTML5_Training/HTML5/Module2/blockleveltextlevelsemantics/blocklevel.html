HTML Elements are Awesome

Yesterday I started to write in Word and realized that everything uses markup to separate content, we simply don’t always see it. For instance, in Word you define the start and end of content; if you want to create a table of contents you define specific content to be for a TOC; you define specific content for figures and page structure. This is the same as HTML5!
